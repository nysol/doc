\section{asin - Inverse Sine\label{sect:asin}}
\index{asin@asin}

Format: asin($num$)

Compute arcsine (inverse sine). The function evaluates the principal value ranges from  $-1.0\sim 1.0$, and returns values within the parameter of $-\pi/2\sim \pi/2$.
