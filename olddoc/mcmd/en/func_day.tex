\section{day - Day\label{sect:day}}
\index{day@day}

Format 1: day($date$)

Format 2: day($time$)

Format 3: days($date$)

Format 4: days($time$)


Extract day from $time$ or $date$. The function shown in format 1 and 2 returns the numeric value, the function in format 3 and 4 returns 2-digit fixed-length string.
