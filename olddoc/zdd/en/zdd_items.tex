
\section{items : Aggregate weight of all items in ZDD\label{sect:items }}
\subsection*{Format}
$obj$.items $\rightarrow$ $zdd$

\subsection*{Description}
Construct a new ZDD object $zdd$ by aggregating the weights from items in ZDD object $obj$.
