NYSOL
========================================

NYSOL is a software library developed to support knowledge discovery from Big Data efficiently and effectively
[#f0]_ .
It has been provided as a set of commands that can be used on shells so far, [#f1]_ 
the **nysol_python** library was developed additionally to make these available on Python.
As of today, nysol_python contains 
MCMD module which demonstrates its power in data preprocessing and 
Take module which provides a variety of tools available for Itemset mining.
